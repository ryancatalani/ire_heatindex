The #IRE17 Heat Index

Why? In the last hour, % of #IRE17 tweets have mentioned the weather in Phoenix. The heat index is calculated based on that. (How?)

The number of tweets that mention certain weather keywords and use the hashtag #IRE17 are divided by the number of tweets that just use #IRE17. This ratio (between 0 and 1) is squared and multiplied by 100. Since this value is between 0 and 100, it's sort of like the Celsius scale (which is based on 0° as the freezing point of water and 100° as the boiling point). So this value is then "converted" to Fahrenheit (by multiplying by 1.8 and adding 32), then rounded. See the code on GitHub.
